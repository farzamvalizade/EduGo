# EduGo

## 1. معرفی پروژه

### نام پروژه

EduGo

### هدف پروژه

EduGo یک سامانه آموزشی مرحله‌به‌مرحله است که با هدف ایجاد یک مرجع جامع و ساختاریافته برای یادگیری طراحی شده است. در این سامانه، دانش‌آموزان می‌توانند مطالب آموزشی را به‌صورت گام‌به‌گام فرا گرفته و پس از هر مرحله به پرسش‌های مرتبط پاسخ دهند.

هدف اصلی پروژه، رفع کمبود منابع آموزشی یکپارچه و ایجاد بستری برای یادگیری تدریجی و هدفمند است.

### گروه هدف

- دانش‌آموزان
- علاقه‌مندان به یادگیری مرحله‌ای
- مراکز آموزشی و تولیدکنندگان محتوای آموزشی

# 5. ساختار آموزشی

ساختار آموزشی سامانه به صورت زیر است:

درس
→ مبحث
→ مرحله
→ سوال

هر مرحله شامل تعدادی سوال آموزشی است.

کاربر برای عبور از هر مرحله باید حد نصاب تعیین‌شده توسط تولیدکننده محتوا را کسب کند.

مثال:

- تعداد سوالات: 5
- حد نصاب قبولی: 4 پاسخ صحیح

در این حالت کاربر برای قبولی باید حداقل به 4 سوال پاسخ صحیح بدهد.

---

# 6. سیستم امتیازدهی

برای هر پاسخ صحیح، کاربر مقداری XP دریافت می‌کند.

میزان XP هر سوال توسط سیستم تعیین می‌شود و در پروفایل کاربر ذخیره می‌گردد.

XP جهت نمایش میزان فعالیت و پیشرفت کاربر استفاده می‌شود.

# 8. الگوریتم پیشروی آموزشی

1. کاربر یک مبحث را انتخاب می‌کند.
2. سیستم آخرین مرحله حل شده را بررسی می‌کند.
3. مرحله بعدی نمایش داده می‌شود.
4. کاربر به سوالات پاسخ می‌دهد.
5. نمره مرحله محاسبه می‌شود.
6. در صورت کسب حد نصاب، مرحله بعد باز می‌شود.
7. در غیر این صورت کاربر همان مرحله را مجدداً انجام می‌دهد.

# نحوه اجرای بک‌اند

دستورات زیر را به ترتیب در روت پروژه اجرا کنید. پایتون روی سیستم نصب باشد.

```sh
cd backend/
python -m pip install -r req.txt
python manage.py runserver
```

---

# نحوه اجرا فرانت‌اند

دستورات زیر به ترتیب در روت پروژه اجرا کنید. نیاز به Node.js دارید.

```sh
cd frontend/
npm install
npm run dev
```
